Rodzaje Pętli

Pętla for

Konstrukcja:

for (inicjalizacja; warunek; zmiana) {
}

Opis: Pętla for używana jest, gdy znana jest liczba powtórzeń. Składa się z trzech części: inicjalizacji, warunku oraz inkrementacji/dekrementacji.

Pętla while (z kontrolowanym wejściem)

Konstrukcja:

while (warunek) {
}

Opis: Pętla while wykonuje kod dopóki warunek jest prawdziwy. Sprawdza warunek przed każdym wykonaniem – jeśli jest fałszywy od początku, pętla się nie wykona.

Schemat blokowy:

Pętla do...while (z kontrolowanym wyjściem)

Konstrukcja:

do {  
} while (warunek);

Opis: Pętla do while zawsze wykona się co najmniej raz, ponieważ warunek sprawdzany jest po pierwszym wykonaniu kodu.
